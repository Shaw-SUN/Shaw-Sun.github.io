# 媒体查询 @media *

浏览器窗口符合一定条件实现某些样式

@media screen and (*: *) and (*: *) ... {
  selector {
  }
}
1
2
3
4

利弊
优点：一套页面适应多端设备，提升开发效率
缺点：页面效果不如单独为一终端定制的页面；性能问题；维护成本高
总结：大部分项目不会整体采用响应式

# 移动端单位

  • 绝对单位

    • px: 像素

  • 相对单位

    • em: 父级元素font-size:值的倍数(如父级没有则向上搜索)
    • rem: <html>标签font-size:值的倍数；
    • vw: 相对于视口*宽度的 1%
    • vh: 相对于视口*高度的 1%
    • *百分比法(过于繁琐)

用 js 设置 设计稿像素除以基数
